# Multi search
This addon enable us to search to a specific site.

#### How to launch?

- Open Browser
- Type: *ms {keyword} {query}*
- Hit enter
- Example: **ms t imsriramb**

Here, the keyword denotes the site that we are going to search.

##### Key words and Values

- t - Twitter
- g - Google
- gh - Github
- b - Bing
- d - DuckDuckGo
- y - Youtube
- ya - Yandex
- wp - Wordpress
- a - Amazon
- s - Stack Overflow
- dp - Dogpile
